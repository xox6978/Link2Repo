## ⚙️ راهنمای نصب و راه‌اندازی (فقط برای بار اول)

برای استفاده از این پروژه روی حساب خود، مراحل زیر را یک‌بار انجام دهید:

### مرحله ۱: ساخت نسخه شخصی (Fork)
1. روی دکمه **Fork** در بالای صفحه کلیک کنید.
2. نام دلخواه انتخاب کرده و **Create fork** را بزنید.

<br/>

### مرحله ۲: تنظیم دسترسی‌ها
1. وارد تب **Settings** مخزن خود شوید.
2. از منوی سمت چپ، به **Actions → General** بروید.
3. به بخش **Workflow permissions** بروید.
4. گزینه **Read and write permissions** را انتخاب و ذخیره کنید.

<br/>

### مرحله ۳: فعال‌سازی Actions
1. به تب **Actions** بروید.
2. روی دکمه:
   `I understand my workflows, go ahead and enable them`
   کلیک کنید.

<br/>

### مرحله ۴ (اختیاری): تنظیم یوتیوب
برای دانلود از یوتیوب، ممکن است نیاز به ثبت کوکی مرورگر داشته باشید:

1. افزونه `Get cookies.txt LOCALLY` را نصب کنید:
   - Chrome: https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc  
   - Firefox: https://addons.mozilla.org/en-US/firefox/addon/get-cookies-txt-locally/
2. وارد YouTube شوید و لاگین کنید.
3. از طریق افزونه، گزینه **Export as** را بزنید و فایل را ذخیره کنید.
4. محتوای فایل را کپی کنید.
5. در گیت‌هاب مسیر زیر را باز کنید:  
   **Settings → Secrets and variables → Actions**
6. روی **New repository secret** کلیک کنید.
7. مقدار زیر را به‌عنوان Name وارد کنید:

```text
YOUTUBE_COOKIES
```

8. محتوای فایل را در بخش Secret قرار داده و ذخیره کنید.

## 📋 راهنمای استفاده

### دانلود فایل
1. به تب **Actions** بروید.
2. workflow دانلود را انتخاب کنید.
3. روی **Run workflow** کلیک کنید.
4. تنظیمات:
   - لینک دانلود: وارد کردن یک یا چند لینک
   - کیفیت: مخصوص یوتیوب (پیشنهاد: 480p یا 720p)
   - رمز: اختیاری
5. اجرای workflow و انتظار تا تکمیل شدن آن

<br/>

### دریافت فایل

#### روش سریع
- فایل **Links.md** را در تب Code باز کنید و روی لینک‌ها کلیک کنید.

#### روش معمولی
1. وارد پوشه **dl** شوید.
2. فایل موردنظر را باز کنید.
3. از منوی سه‌نقطه:
   - گزینه **Download** برای دانلود مستقیم
   - یا **View → Copy link address** برای لینک خام
